All about data-layout

You can change the layout of the shopping bag by adding the parameter data-layout to the "ec-cart-widget" container in the integration code.

<data-layout="VALUE">

Where VALUE is one of the supported values for the data-layout parameter:

  • SMALL_ICON
  • SMALL_ICON_COUNTER
  • COUNTER_ONLY
  • TITLE_COUNTER
  • MEDIUM_ICON_COUNTER
  • MEDIUM_ICON_TITLE_COUNTER
  • BIG_ICON_TITLE_SUBTOTAL
  • BIG_ICON_DETAILS_SUBTOTAL

You can also use <data-show-empty-cart="VALUE"> with TRUE or FALSE as value for show or hide the shopping bag when it’s empty.

Or you can change the default shopping bag icon <data-icon="VALUE> where values might be BAG, CART or BASKET.

Or you can change the shape of the border around the shopping bag <data-fixed-shape="VALUE"> where values might be NON, RECT or PILL.

Or you can change the shopping bag position <data-fixed="VALUE"> <data-fixed-position="VALUE"> <data-horizontal-indent="VALUE"> <data-vertical-indent="VALUE"> where values for data-fixed might be TRUE or FALSE, for data-fixed-position

  • TOP_LEFT
  • TOP_RIGHT
  • BOTTOM_LEFT
  • BOTTOM_RIGHT
and values for data-horizontal-indent and data-vertical-indent is a positive integer including zero.

Or you can disable responsiveness of the shopping bag widget to make it look the same in different screen sizes, desktop and mobile with <data-responsive="VALUE">. Use TRUE value and the cart will automatically adapt to different screen sizes, or FALSE and the cart will have the same design on different screen sizes.

Finally, you can use a custom shopping bag icon on your website with <data-responsive="VALUE">. Where value is URL to your image or SVG markup.
